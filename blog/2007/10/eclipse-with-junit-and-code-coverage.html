Eclipse with JUnit and Code Coverage tool

2007-10-04

Tags: 程式語言

我針對利用Eclipse IDE寫出來的一堆Java Code總是喜歡利用Eclipse JUnit plugin跟TPTP Code Coverage plugin去作測試，順便分析一下Code Coverage，找出程式中的盲點。

不過...這儿天卻發現TPTP Code Coverage plugin如果遇到Java Code中有JDK 5.0以上的語法，在Eclipse上就會出現怪怪的問題，出現Error。

後來想想，現在使用的這個TPTP Code Coverage plugin在官網上也已經列為Deprecated項目了，所以還是找找有沒有其它相同功能的Eclipse Code Coverage plugin比較實在。

在拜請過「股溝」大神後，找到的是 EclEmma這個不少人用的plugin。試用過後發現還真是不錯用，TPTP Code Coverage提供的功能它都有，而且跑起來還比TPTP Code Coverage要快些，所以當下就決定很沒忠誠的改用這東東了...XD

IBM developerWorks有篇簡介這工具的文章，有興趣的人可以 按此過去看看...:)
