Design Patterns in Java

2005-02-17

Tags: 程式語言

Design Patterns其實也不是啥新東西了，也不是只有Java才能用這鍋東東，C++、.Net上面也用了不少這東西。其實...平常可能就已經用到Design Patterns的概念，只是你我不自知而已。如果你也是Java一族的話，下面這二本可以參考一下，我覺的它寫的還蠻容易懂的，至少不會看個二行就想去 找周公下棋了。

1.「Design Patterns於Java語言上的實際應用」

這本是小日本仔寫的，台灣出的是中譯本，翻譯的還算可以，書中的例子也還蠻生活化的。

2.「Head First Design Patterns」

這本是前陣子才出的熱門書，書中用很多的圖例跟好笑的例子來解釋Design Patterns的概念，看起來就像在看圖畫書一樣，蠻有趣的。如果沒有英文恐懼症的，可以試試這本。
