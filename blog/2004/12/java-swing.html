Java Swing

2004-12-29

Tags: 程式語言

從來沒想過自已會需要研究Swing的東西，結果還是碰上了。也正因為需要，又去天瓏敗家了...-_-|||找來找去...嘿!突然發現中文書中有一本好貨，而且這本中文書還是全彩的。彩色的看起來就是爽度大增。

書名叫「精通Java Swing程式設計」(不過書中有些小錯誤，記的要 下戴刊誤，自己手動更正一下)，是一本很不錯入門書籍。裡面有大量的Sample Code跟執行時的GUI圖片，讓人比較會想學下去，而不會想找姜太公下棋。


除了這本之外，還有二本好貨一本是Manning出的「Swing, Second Edition」( 按我看簡介)，另一本是O'Reilly的「Java Swing, Second Edition」( 按我看簡介)，這兩本都是很棒的進階書籍。這兩本也都有中譯本，至於翻譯的好不好，小弟就不敢掛保証了。
